This repository contains a small utility to synthesize daily construction material
prices for testing and demonstration.

Files:
- `generate_data.py`: Generates synthetic daily prices and writes a CSV.
- `construction_material_prices.csv`: (may be generated by the script)

Usage:

PowerShell (Windows):
```pwsh
python .\generate_data.py --output construction_material_prices.csv
```

Options:
- `--output` / `-o`: Path to output CSV (default: `construction_material_prices.csv`).
- `--days` / `-d`: Number of days of history to generate (default: 365).
- `--seed`: Random seed for reproducibility (default: 42).

If you create a GitHub repository for this folder, commit the code and exclude
local virtual environments and large data files via `.gitignore` (provided).
